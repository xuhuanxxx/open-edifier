# 研究资料

逆向笔记和原始观察应放在这里。禁止提交专有 APK、固件镜像或其他属于厂商的二进制文件。

公开研究必须经过脱敏并且能够复现。优先使用协议表、合成帧、确定性 mock 和固件/型号标识，不要使用原始抓包。禁止包含设备 IP 地址、MAC 地址、Wi-Fi 名称、已配对设备记录、厂商手册、APK、DEX 文件或解包后的原生库。

在过时脚本得到统一处理前，不得把探索性假设提升为协议文档。S260 二进制分帧最终确认的一字节长度实现在 `docs/aaec.md` 中说明。

首轮 S260 研究的脱敏纪实见 [`docs/research-journey.md`](../docs/research-journey.md)。它保留实验方法，但不会保存原始抓包或专有产物。
